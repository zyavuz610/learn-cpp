__ linux commands __
---------------------
FILE COMMANDS
List files in the directory:
	ls
	...
	ls
	ls -l
	ls -a ; shows all files including hidden files 
Show directory you are currently working in:
	pwd
Create a new directory:
	mkdir [directory]
	...
	mkdir "dizin"
Remove a file:
	rm [file_name] 
	...
	rm "dosya" ; 
	rm "dosya" "dosya2" ... ; birden çok dosya sil
	rm -r "dizin" ; dizin içindekileri de sil
	rmdir "dizin" ; boş dizini siler
Remove a directory recursively:
	rm -r [directory_name]
Recursively remove a directory without requiring confirmation:
	rm -rf [directory_name]	
Copy the contents of one file to another file:
	cp [file_name1] [file_name2]
Recursively copy the contents of one file to a second file:
	cp -r [directory_name1] [directory_name2]
	...
	cp "kaynak" "hedef"
	cp dosya1.txt dosya2.txt dosya3.txt yeni_dizin/
	cp -r dizin1 dizin2 ; alt dizinler dahil kopyalar
	cp -r * yedek/
Rename [file_name1] to [file_name2] with the command:
	mv [file_name1] [file_name2]
	...
	mv src dest
	mv dosyam.txt belgeler/
	mv eski_ad.txt yeni_ad.txt ; rename
	mv proje/ yedekler/; move directory
	mv dosya1.txt dosya2.txt dosya3.txt yeni_dizin/; moving multiple files
Create a symbolic link to a file:
	ln -s /path/to/[file_name] [link_name]
Create a new file using touch:
	touch [file_name]
	...
	touch "dosya" ; dosya oluştur veya tarih değiştir
Show the contents of a file:
	more [file_name]
or use the cat command:
	cat [file_name]
Append file contents to another file:
	cat [file_name1] >> [file_name2]
Display the first 10 lines of a file with head command:
	head [file_name]
Show the last 10 lines of a file with tail command:
	tail [file_name]
Show the number of words, lines, and bytes in a file using wc:
	wc
Compare two files and display differences:
	diff [file1] [file2]
Store the command output in a file and skip the terminal output:
	[command] > [filename]

DIRECTORY NAVIGATION
Move up one level in the directory tree structure:
	cd ..
Change directory to $HOME:
	cd
Change location to a specified directory:
	cd /chosen/directory
Sample:
	cd "dizin"
	cd .. ; bir üst dizin
	cd / ; kök dizin ya da c:\
